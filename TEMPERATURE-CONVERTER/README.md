Temperature Converter

Description
This project is a Temperature Converter Web Application developed as Task 3 of the Web Development Internship at Oasis Infobyte.  
It converts temperature values between Celsius, Fahrenheit, and Kelvin.

Technologies Used
HTML  
CSS  
JavaScript  

Features
Converts temperatures between multiple units
Interactive and user-friendly interface
Instant result display
Responsive design

Learning Outcomes
Strengthened JavaScript logic and calculations
Hands-on experience with event handling
Improved form input handling and validation
Better understanding of dynamic UI updates

Conclusion
This task enhanced my problem-solving skills and practical knowledge of JavaScript.
