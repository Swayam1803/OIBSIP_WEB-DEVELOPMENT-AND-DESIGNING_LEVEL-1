Workshop Management System - Web Application


Developed a full-stack web application to manage and display upcoming workshops and collect participant feedback. Built front-end using HTML, CSS and JavaScript, where users can view scheduled upcoming workshops and provide feedback on attended sessions through a user friendly interface. Implemented back-end using MongoDB to store workshop details and feedback data. Designed features including adding new workshops, listing upcoming events, feedback submission and ensuring seamless data storage and retrieval. Demonstrated skills in web development, form handling, database schema design, and full-stack integration between front-end and data storage. Project helped to simulate a real-world workshop management facilitating easy addition of new workshops and collection of feedback

Amazon Clone - Landing Page


Developed a responsive Amazon homepage clone to replicate the core layout, design, and user interface of the Amazon e-commerce platform. The project focuses on recreating a real-world website experience using modern frontend web technologies while maintaining clean code structure and responsive design principles. Fully responsive homepage layout with header, navigation bar, product sections, and footer. Product cards with images, pricing, and descriptions styled to match the original interface. Search bar and navigation menu UI for improved user experience. Hover effects and interactive elements to enhance usability. Optimized layout for desktop and mobile screens.
